# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in the Doctor Appointment System, please report it by creating an issue on this repository or by contacting the maintainer directly.

## Security Best Practices Implemented

- Password hashing using bcrypt
- JWT-based authentication
- Input validation and sanitization
- CORS configuration
- Environment variable management
- Rate limiting on API endpoints
