# Frequently Asked Questions

## General Questions

### Q: How do I book an appointment?
A: Register as a patient, search for a doctor by specialization, and select an available time slot.

### Q: Can I cancel my appointment?
A: Yes, you can cancel appointments from your dashboard up to 24 hours before the scheduled time.

### Q: Is my data secure?
A: Yes, we use JWT authentication, password hashing with bcrypt, and secure HTTPS connections.

## Technical Questions

### Q: What tech stack is used?
A: React + Vite for frontend, Node.js + Express for backend, and MongoDB Atlas for database.

### Q: How do I run this project locally?
A: Clone the repo, install dependencies with npm install, and run npm run dev for frontend.

### Q: Where is the project deployed?
A: Frontend on Vercel, backend on Render, database on MongoDB Atlas.
